# Layout Skill

- Goal:
  - Build or review page shells that match the shared UI structure.
- Required checks:
  - Navbar shows Chat Hub, Marketplace, Discover New, Agents.
  - Language selector exists.
  - Sign-in entry exists.
  - Light theme is preserved.
- Communication:
  - Escalate missing API or route dependencies using the shared communication contract.
- Chat Hub rule:
  - Keep left sidebar, center chat, right insights panel on desktop.
- Marketplace rule:
  - Keep top chips and left sidebar filters visible in the layout.
