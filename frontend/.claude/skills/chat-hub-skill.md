# Chat Hub Skill

- Goal:
  - Build or refine the Chat Hub experience.
- Must include:
  - 400+ model-capable left sidebar structure.
  - Center chat area.
  - Suggested prompts section.
  - Prompt guide panel.
  - Active model shown near composer.
  - Right sidebar with stats, usage, chart, quick actions.
- Interaction rules:
  - Support text plus voice and file entry points.
  - Model switching must update visible state cleanly.
- Communication:
  - Tell backend when chat needs model metadata, usage stats, or composer-related API support.
