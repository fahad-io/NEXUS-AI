# Analyzer Workflow

- Preparation:
  - Load HTML reference and shared UI specs.
  - Focus on structure and behavior parity.
- Review areas:
  - Home and navbar.
  - Chat Hub layout and panels.
  - Marketplace layout and filters.
  - Discover New and Agents entry points.
- Checklist:
  - Navbar labels match reference.
  - Language selector exists.
  - Sign-in entry exists.
  - Chat Hub keeps 3-column desktop structure.
  - Left chat sidebar supports a large model catalog.
  - Center chat area includes suggested prompts and prompt guidance.
  - Active model is visible near the chat box.
  - Right panel includes stats, usage, chart, quick actions.
  - Marketplace chips and sidebar filters exist.
  - Model cards expose chat CTA.
- Output:
  - List mismatches by page.
  - Separate structural issues from visual polish gaps.
