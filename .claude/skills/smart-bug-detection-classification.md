# Smart Bug Detection & Classification Skill

## Purpose
Detect, classify, and structure bugs from test execution results.

---

## Responsibilities
- Detect failures from API responses or logs
- Classify bug severity:
  - Blocker
  - Critical
  - Major
  - Minor
- Identify root cause hints
- Map bug to API/test case

---

## Bug Categories
- Functional bug
- Security bug
- Performance issue
- Validation issue
- Integration failure

---

## Output Fields
- bug_title
- severity
- endpoint
- steps_to_reproduce
- expected_result
- actual_result
- suspected_cause
