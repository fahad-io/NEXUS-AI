# Security Penetration API Skill

## Purpose
Perform advanced security-focused QA on APIs.

---

## Responsibilities
- Detect authentication bypass risks
- Test role escalation scenarios
- Identify injection points:
  - SQL
  - NoSQL
  - Command injection (basic patterns)
- Check IDOR (Insecure Direct Object Reference)
- Validate sensitive data exposure

---

## Security Test Types
- Unauthorized access attempts
- Token manipulation
- Parameter tampering
- Privilege escalation
- Rate limit abuse

---

## Rules
- Be aggressive but safe
- Focus on real-world attack patterns
- Never assume API is secure
