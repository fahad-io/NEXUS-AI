# Security Validation Skill

## Purpose
Add basic security checks in API automation tests.

---

## Responsibilities
- Check unauthorized access responses
- Validate missing token scenarios
- Detect forbidden access cases (403)
- Ensure sensitive data is not exposed

---

## Rules
- Never assume API is secure
- Always include at least basic auth test coverage
- Validate role-based access if applicable
