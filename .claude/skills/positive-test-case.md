# Positive Test Case Skill

## Purpose
Generate valid and successful API test cases.

---

## Responsibilities
- Create valid request payloads
- Include correct authentication
- Ensure expected success responses
- Cover standard business flows

---

## Rules
- Use only valid inputs
- Must return success status codes (200, 201, etc.)
- Ensure realistic API usage scenarios
- No invalid or broken inputs
