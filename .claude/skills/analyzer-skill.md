# Analyzer Skill

- Goal:
  - Compare app behavior against the HTML reference and shared specs.
- Compare:
  - Navbar labels and controls.
  - Chat 3-column layout.
  - Marketplace chips and sidebar filters.
  - Model-card CTA and right-panel insights.
- Review style:
  - Focus on structure and behavior first.
  - Flag missing elements before visual polish gaps.
  - Report mismatches with the shared communication contract.
- Output:
  - Mismatches by page.
  - Actionable follow-up items only.
