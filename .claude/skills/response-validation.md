# Response Validation Skill

## Purpose
Validate API responses against expected structure.

---

## Responsibilities
- Validate status codes
- Validate response schema
- Check required fields in response
- Validate error message consistency
- Ensure data types are correct

---

## Rules
- Must match request logic
- Detect missing or extra fields
- Ensure consistent API contracts
