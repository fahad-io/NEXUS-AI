# Test Structure Organization Skill

## Purpose
Organize Playwright test cases into clean, readable structure.

---

## Responsibilities
- Group related test cases
- Create meaningful test names
- Avoid duplication
- Ensure modular test structure

---

## Best Practices
- One test = one scenario
- Use descriptive names
- Keep tests independent
- Avoid shared state
