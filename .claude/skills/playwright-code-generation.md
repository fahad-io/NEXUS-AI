# Playwright Code Generation Skill

## Purpose
Convert API test cases into clean, executable Playwright test scripts.

---

## Responsibilities
- Convert structured test cases into @playwright/test format
- Ensure proper request handling using API request context
- Generate clean, maintainable test code

---

## Rules

- Must be production-ready code

- No explanations
- No comments outside code logic
