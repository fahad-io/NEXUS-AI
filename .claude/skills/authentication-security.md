# Authentication & Security Skill

## Purpose
Validate API security and authentication mechanisms.

---

## Responsibilities
- Validate token-based authentication
- Check role-based access control
- Test expired tokens
- Test missing authorization headers
- Detect privilege escalation risks
- Basic injection attempts (SQL, NoSQL, XSS patterns)

---

## Rules
- Always assume system may be insecure
- Try unauthorized access scenarios
- Validate strict access control enforcement
