# Edge Case Skill

## Purpose
Generate boundary and extreme condition test cases.

---

## Responsibilities
- Empty values
- Null values
- Very large inputs
- Special characters
- Unicode inputs
- Boundary limits (min/max)
- Duplicate requests
- High frequency calls

---

## Rules
- Focus on breaking the system safely
- Identify weak validation areas
- Test system limits and constraints
