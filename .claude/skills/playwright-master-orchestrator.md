# Playwright Master Orchestrator Skill

## Purpose
Control and coordinate all Playwright automation skills.

---

## Responsibilities
You must orchestrate:

1. API Request Builder Skill
2. Code Generation Skill
3. Assertion Engine Skill
4. Error Handling Skill
5. Security Validation Skill
6. Execution Stability Skill
7. Test Structure Skill

---

## Workflow Order

STEP 1: Understand test case  
STEP 2: Build API request  
STEP 3: Generate Playwright test block  
STEP 4: Add assertions  
STEP 5: Add negative validations  
STEP 6: Ensure stability rules  
STEP 7: Output final code

---

## Output Rule
- ONLY Playwright code
- No explanation
- No JSON
