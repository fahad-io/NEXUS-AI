# UI Rules

- Use the uploaded HTML file as the structure reference.
- Navbar must include:
  - Chat Hub
  - Marketplace
  - Discover New
  - Agents
- A language selector must exist in the navbar.
- A sign-in button or entry point must exist in the navbar.
- Keep a modern light theme.
- Do not switch the product to dark mode.
- Chat Hub must use a 3-column desktop layout:
  - Left model sidebar.
  - Center chat workspace.
  - Right model insights panel.
- Marketplace must include top chips filters:
  - language
  - vision
  - code
  - image gen
  - audio
  - open source
- Marketplace must include left sidebar filters:
  - provider
  - pricing
  - rating
  - token limit slider
- Model cards must expose a primary CTA to enter Chat Hub.
