# Fix Command

- Goal:
  - Resolve verified issues with the least risky edits.
- Steps:
  - Reproduce the issue.
  - Fix the smallest responsible layer.
  - Re-run relevant lint, test, or build commands.
  - Re-check route and UI behavior if user-facing.
- Rules:
  - Do not rewrite unrelated files.
  - Do not leave placeholder fixes.
  - Document any remaining blockers clearly.
