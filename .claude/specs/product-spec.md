# Product Spec

- Purpose:
  - A light-themed AI model hub to discover, compare, and use models in one place.
- Main pages:
  - Home
  - Chat Hub
  - Marketplace
  - Discover New
  - Agents
  - Login
  - Signup
- Core user flows:
  - Guests can browse, compare, and start short-lived chat sessions.
  - Authenticated users can keep history, access protected dashboard features, and migrate guest work.
- Model experience:
  - Users can switch active models from chat and marketplace entry points.
  - Active model status is visible near the chat composer.
- Media inputs:
  - Voice, file, and video-related UI entry points must be present.
  - Supported UI should guide users toward compatible models.
- Product promise:
  - Help users choose the right model quickly, then continue directly into Chat Hub.
