# Marketplace Spec

- Top chips filters:
  - language
  - vision
  - code
  - image gen
  - audio
  - open source
- Left sidebar filters:
  - provider
  - pricing
  - rating
  - token limit slider
- Model cards:
  - Show model name, provider, capability tags, rating, and pricing signal.
  - Include primary CTA: "Use in Chat Hub".
- Behavior:
  - Filters update results without breaking navigation.
  - CTA opens Chat Hub with the chosen model already active.
  - Marketplace supports both browsing and task-based comparison.
